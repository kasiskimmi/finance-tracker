# 💰 Personal Finance Tracker & Expense Analyzer

A feature-rich command-line app to manage your personal finances built with Python.

## ✨ Features
- Track income & expenses by category
- Set monthly budgets with visual progress bars
- Monthly & yearly analytics
- Spending trend analysis (month-over-month)
- Top expenses ranking
- Keyword search across transactions
- Export data to CSV

## 🛠️ Tech Stack
- Python 3
- JSON (local data storage)
- Colorama (colored CLI output)

## 🚀 How to Run
pip install colorama
python finance_tracker.py

## 📊 Menu Options
1. Add Expense / Income
2. View & filter transactions
3. Set budgets per category
4. Monthly & yearly reports
5. Export to CSV
